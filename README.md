### What I learned

I learned how to correctly embed Google Fonts and use Flexbox for centering elements within a layout.

```html
<link
  href="https://fonts.googleapis.com/css2?family=Outfit:wght@400;700&display=swap"
  rel="stylesheet"
/>
```
